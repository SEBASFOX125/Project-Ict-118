Random Color Generator Project - HTML, CSS, JavaScript

Let's create a tool that generates multiple random colors and codes with JavaScript. Upon refreshing the page, the colors and codes change randomly for up to 30 color containers. The website is also designed to be responsive, adapting to different screen sizes. It displays the colors in four columns on larger screens, then decreases to three, two, and ultimately one column for smaller screens. This provides a seamless user experience on any device.

Below are screenshots of the Random color generator

Technologies Used
  • HTML5 for the structure of the webpage
  • CSS3 for styling the elements and layout
  • JavaScript for dynamically generating random colors and updating the UI
Role and Contributions
  • Front-End Development: Crafted the HTML structure for the color containers and applied CSS to style them, ensuring a clean and responsive design.
  • Responsive Design Implementation: Utilized CSS media queries to make the layout adaptable to different screen sizes, enhancing user experience on both desktop and mobile devices.
  • JavaScript Functionality: Wrote JavaScript functions to generate random hexadecimal color codes and dynamically assign them to the background of each container upon page load.
  • User Experience Enhancement: Ensured that the color codes are displayed alongside their corresponding color blocks, allowing users to easily identify and use the generated colors.
